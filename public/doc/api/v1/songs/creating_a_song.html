Songs API

Creating a song

POST v1/songs

Creates a son

Parameters

Name Description
song[name] Song name
song[duration] Duration of the song in seconds
song[genre] Genre of the song
song[album_id] Album which the song belongs to
song[featured] Song featured (true) or not featured (false). Default value is false
song[here] Image of a featured song
song[description] Description of a featured song

Request

Headers

Accept: application/json
Content-Type: application/json
Host: example.org
Cookie:

POST v1/songs
